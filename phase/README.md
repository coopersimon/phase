# phase

This is a PlayStation emulator. It is in development. It does software rendering.

## Game list

Below is a list of selected games and their status.

- Crash Bandicoot: Boots, rendering issues
- Crash Team Racing: Does not boot.
- Final Fantasy 7: MDEC crash
- Final Fantasy Tactics: Does not boot.
- Gran Turismo: Crashes trying to use nonsense GPU command
- Metal Gear Solid: Waiting for SPU interrupt.
- Resident Evil: MDEC crash
- Resident Evil 2: Does not boot.
- Tony Hawk's Pro Skater 2: Does not boot.
- Wild ARMS: MDEC crash
